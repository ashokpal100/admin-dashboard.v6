# admin-dashboard.v6

setup steps:

git clone https://github.com/ashokpal100/admin-dashboard.v6
cd admin-dashboard.v6
npm install
npm install bower -g
bower install
npm start

#angularjs #bootstrap #gulp #nodejs #expressjs #echarts


## How to run code?

For Production Environment

```
gulp prod
```

For Development Environment

```
gulp dev
```
